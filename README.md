## About this portfolio

This portfolio documents my learning journey through ITAI 4370, from the fundamentals of telecommunications to advanced AI applications in modern networks. I came into the course with no telecom background, so the record here starts at the very beginning, a single signal traveling through a channel, and ends with self-managing network simulations and machine learning models that steer traffic.

Each module has its own folder holding the actual work: assignments, lab notebooks that run and show their output, reports, diagrams, and screenshots. Every module folder has a `README.md` that explains what the module covered, the problem I worked on, the methods and tools I used, the results, and what I learned.

## My progression, from basic to advanced

The course was built as a climb, and the portfolio follows the same arc:

- **Modules 01 to 02, the basics.** What a communication system is, how a signal survives noise, how networks are addressed and layered, and the first look at the 5G core.
- **Modules 03 to 05, AI enters and the architecture deepens.** My first machine learning models for traffic prediction, anomaly detection, and segmentation; modeling 5G slices; and a deep look at Open RAN.
- **Modules 06 to 07, applied and honest.** Comparing forecasting models and catching a target leakage bug; compressing a model to run on a small edge device and fixing a broken distillation step.
- **Modules 08 to 09, the wider picture.** Security and energy, where AI both helps and has to be kept in check.
- **The midterm and beyond (Modules 10, 11, 13, 14).** A three-way network simulation, the 6G vision, using AI responsibly, and the final reflections.

By the end I could read a packet capture, model how far a signal travels, build and compare machine learning models, catch my own mistakes, and simulate a network that manages its own traffic.
