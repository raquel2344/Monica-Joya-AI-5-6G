## Focus of this module

The 6G vision and what makes a network AI-native. 6G aims well past 5G, with targets like terabit-per-second peak speeds, latency near a tenth of a millisecond, and much higher device density, which pushes it into terahertz spectrum.

## What I learned

I learned that AI-native means the network is self-optimizing and self-healing by design and driven by intent, where an operator states a goal and the network configures itself to meet it. Federated learning fits here because devices can help train a shared model without sending their raw data off, which gives privacy and a better network at the same time.
